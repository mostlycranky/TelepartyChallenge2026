## Features

### Core Functionality
- **Create Chat Rooms** - Start a new private room instantly
- **Join Rooms by ID** - Enter a room ID to join conversations
- **Real-time Messaging** - Send and receive messages via WebSocket
- **User Nicknames** - Custom display names for all participants
- **Message History** - Previous messages load when joining
- **Typing Indicators** - See when others are typing

### Additional Features
- **Emoji Avatars** - Choose from 14 different emoji icons
- **System Messages** - Join/leave notifications
- **Shareable Room ID** - Copy to invite others
- **Connection Status** - Visual WebSocket state indicator
- **Responsive Design** - Works on desktop and mobile
- **Local Storage** - Remembers nickname and avatar
